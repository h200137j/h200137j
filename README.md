## Features

- **Interactive particle canvas** ‚ÄĒ reacts to mouse movement in real time
- **Custom cursor** ‚ÄĒ spring-physics dual-layer cursor with hover expansion
- **Persistent dark/light mode** ‚ÄĒ saved to `localStorage`, respects system preference
- **Smooth scroll navigation** ‚ÄĒ single-page with animated section reveals
- **Project modals** ‚ÄĒ `AnimatePresence` powered overlays with full project details
- **Contact form** ‚ÄĒ Inertia `useForm`, server-side validation, success animation
- **Fully responsive** ‚ÄĒ mobile-first, works on every screen size

## Getting Started

```bash
# Clone the repo
git clone https://github.com/h200137j/calvin-tafadzwa-mashamba.git
cd calvin-tafadzwa-mashamba

# Install PHP dependencies
composer install

# Install JS dependencies
npm install

# Set up environment
cp .env.example .env
php artisan key:generate

# Run migrations
php artisan migrate

# Start development
php artisan serve   # Terminal 1
npm run dev         # Terminal 2
```
